# Nihongo Practice App

Ứng dụng Fullstack hỗ trợ học tiếng Nhật toàn diện (Từ vựng, Ngữ pháp, Hán tự, Luyện thi JLPT & Dong Du).

## Công nghệ sử dụng
- **Frontend:** React (Vite), TailwindCSS, React Router, Ant Design.
- **Backend:** Java Spring Boot, Spring Security (JWT Auth), Hibernate.
- **Database:** PostgreSQL.


## Tính năng chính
- **Học tập tự do:** Xem danh sách Kanji, Flashcard, Ngữ pháp, Từ vựng mà không cần đăng nhập.
- **Quản lý tiến độ:** Hệ thống đăng ký / đăng nhập bảo mật bằng JWT (JSON Web Tokens) với cơ chế Refresh Token mượt mà tự động giữ trạng thái đăng nhập.
- **Giao diện:** Thiết kế Glassmorphism hiện đại, tối giản và mượt mà.
